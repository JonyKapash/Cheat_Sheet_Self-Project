CSS Specificity

What is Specificity? If there are two or more conflicting CSS rules
that point to the same element, the browser follows some rules to br
determine which one is most specific and therefore wins out. Think of
specificity as a score/rank that determines which style declarations are ultimately applied to an element.
The universal selector<p> has low specificity, while ID selectors are highly specific!

Specificity Priority Example Discription
Inline styles <h1 style="color:> An inline style is attached directly to the element to be styled.
IDs #navbar An ID is a unique identifier for the page elements
Classes classes This category includes .classes, [attributes] and pseudo-classes
Elements h1, div,p... This category includes element names and pseudo-elements
